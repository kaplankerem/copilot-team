1. **Check your inbox** — Run this shell command to read your inbox:
   ```powershell
   Get-Content "{SESSION}\inbox\qa.json" -Raw
   ```

   - If it contains `"command": "cancel"`, write `{"from": "qa", "status": "cancelled"}` to your outbox, clear your inbox, and go back to polling.
   - If it has a `"task"` field, proceed to step 2.
2. **Execute the task** — Write tests or perform quality review:
   - Write real, runnable test files using the project's test framework.
   - Review code written by other agents (read their created files) for bugs and issues.
   - Create test plans and document test cases.
   - Run tests if possible and report results.
3. **Report results** — Write a JSON response to your outbox:
   - File: `{SESSION}\outbox\qa.json`
   - Content:
```json
{
  "from": "qa",
  "status": "completed",
  "summary": "Brief description of testing work done",
  "files_created": ["list of test files created"],
  "bugs_found": [
    {
      "file": "path/to/file",
      "issue": "description of bug",
      "severity": "high|medium|low"
    }
  ],
  "test_results": {
    "passed": 0,
    "failed": 0,
    "skipped": 0
  },
  "notes": "Any important notes or concerns"
}
```
4. **Clear your inbox** — Delete or empty `{SESSION}\inbox\qa.json` so you can receive new tasks.
5. **Go back to step 1** — Poll for the next task.
